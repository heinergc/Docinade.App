## ⚠️ Errores Comunes a Evitar

### ❌ ERROR 1: Columna de descripción con nombre incorrecto

**Incorrecto**:
```
| Criterio | Detalle | Nivel 1 | Nivel 2 |
```

**Correcto**:
```
| Criterio | Descripción | Nivel 1 | Nivel 2 | 
```

---

### ❌ ERROR 2: Descripción en la misma celda que el criterio

**Incorrecto**:
```
| Criterio de Evaluación | Nivel 1 | Nivel 2 |
| Ortografía - Uso correcto de reglas | Val1 | Val2 |
```

**Correcto**:
```
| Criterio | Descripción | Nivel 1 | Nivel 2 |
| Ortografía | Uso correcto de reglas ortográficas | Val1 | Val2 |
```

---

### ❌ ERROR 3: Falta la fila de encabezados

**Incorrecto**:
```
| Ortografía | Desc1 | Val1 | Val2 |
| Coherencia | Desc2 | Val3 | Val4 |
```

**Correcto**:
```
| Criterio | Descripción | Nivel 1 | Nivel 2 |
| Ortografía | Desc1 | Val1 | Val2 |
| Coherencia | Desc2 | Val3 | Val4 |
```

## 📏 Especificaciones Técnicas

### Límites de Campos

| Campo | Límite | Comportamiento |
|-------|--------|----------------|
| **Nombre del Criterio** | 200 caracteres | Se trunca automáticamente con "..." |
| **Descripción** | Sin límite | Se guarda completo |
| **Valores de niveles** | ~1000 caracteres | Depende de la BD |

### Formato de Celdas

- **Texto plano**: Preferible para evitar problemas de formato
- **Sin fórmulas**: Los valores deben ser texto directo
- **Codificación**: UTF-8 para caracteres especiales (tildes, ñ)

### Validaciones del Sistema

El sistema verifica automáticamente:
- ✅ Que exista al menos 1 criterio
- ✅ Que existan al menos 2 niveles de calificación
- ✅ Que la columna "Descripción" esté correctamente nombrada
- ✅ Que no existan filas completamente vacías

## 🎯 Mejores Prácticas

### 1. Nombres Descriptivos para Niveles

**Recomendado**:
- Excelente, Muy Bueno, Bueno, Regular, Insuficiente
- Nivel 4, Nivel 3, Nivel 2, Nivel 1
- Avanzado, Intermedio, Básico, Iniciado

**Evitar**:
- A, B, C, D (muy ambiguo)
- 1, 2, 3, 4 (confunde con puntos)

### 2. Descripciones Claras y Específicas

**Buena descripción**:
> "Uso correcto de reglas ortográficas incluyendo tildes, mayúsculas, puntuación y separación silábica según normas de la RAE"

**Descripción pobre**:
> "Ortografía"

### 3. Valores de Niveles Observables

**Bueno**:
> "Sin errores ortográficos en todo el documento"

**Malo**:
> "Excelente ortografía" (subjetivo)

### 4. Consistencia en la Escala

Si usas números para un nivel, úsalos para todos:
- ✅ Nivel 4, Nivel 3, Nivel 2, Nivel 1
- ❌ Nivel 4, Bueno, 2, Malo

## 🚀 Proceso de Importación

### Pasos en RubricasApp

1. **Ir a Importar Rúbrica**: Menú principal > Rúbricas > Importar desde Excel
2. **Seleccionar archivo Excel**: Buscar y subir el archivo .xlsx
3. **Ingresar datos**:
   - Nombre de la rúbrica
   - Descripción de la rúbrica (opcional)
4. **Hacer clic en "Importar"**
5. **Verificar resultados**: El sistema muestra un resumen de lo importado
6. **Revisar rúbrica**: Se redirige automáticamente a los detalles de la rúbrica

### Logs de Diagnóstico

El sistema registra cada paso:
```
✅ Columna 'Descripción' encontrada en índice 1 (0-based)
⏭️ Saltando columna 'Descripción' en índice 1 al procesar niveles
📝 Descripción encontrada para criterio 'Ortografía': 'Uso correcto...'
Items de evaluación creados: 5
```
