¡Bienvenido(a) a la documentación oficial de **RubricasApp**!

**RubricasApp** es una plataforma web diseñada especialmente para docentes del **Ministerio de Educación Pública (MEP)** de Costa Rica. Su objetivo principal es simplificar y optimizar las tareas administrativas y académicas del día a día, brindando una herramienta moderna, intuitiva y adaptada a la realidad educativa costarricense.

Este sistema permite gestionar de forma eficiente:

- La creación y administración de **instituciones, grupos y materias**
- El registro de **estudiantes** por grupo
- La planificación y evaluación académica con **rúbricas detalladas**
- El control de **asistencia**, incluyendo ausencias, llegadas tardías y reportes
- La gestión del **módulo de conducta** según REA 40862-V21 del MEP
- La generación de **reportes completos** en PDF y Excel, tanto por grupo como por estudiante
- Un robusto **sistema de permisos** para control de acceso granular

Además, **RubricasApp** está en constante evolución, incorporando mejoras basadas en las necesidades reales del sistema educativo costarricense, lo que garantiza una herramienta práctica y centrada en la experiencia del usuario.

## Características principales

### Sistema de Permisos Avanzado

RubricasApp cuenta con un **sistema de 168 permisos** distribuidos en módulos, permitiendo un control granular de acceso según roles y responsabilidades.

### Cumplimiento Normativo MEP

El sistema cumple con las regulaciones del MEP de Costa Rica, incluyendo:
- **Módulo de Conducta** conforme al REA 40862-V21
- **Estructura de niveles educativos** del sistema costarricense
- **Datos geográficos** de las 7 provincias, 82 cantones y 488 distritos

### Reportes Profesionales

Genera reportes académicos, de conducta y asistencia en formatos PDF y Excel listos para presentación oficial.

### Tecnología Moderna

Construido con ASP.NET Core 8.0, SQL Server y las mejores prácticas de desarrollo web para garantizar seguridad, rendimiento y escalabilidad.
