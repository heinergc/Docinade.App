**RubricasApp** es un sistema de gestión educativa diseñado para el **Ministerio de Educación Pública (MEP)** de Costa Rica.

## Características Principales

<CardGrid stagger>
  <Card title="👥 Gestión de Grupos" icon="puzzle">
    Organiza y administra grupos educativos, asigna estudiantes y materias con facilidad.
  </Card>
  
  <Card title="🎓 Estudiantes" icon="open-book">
    Registra y gestiona información completa de estudiantes, incluyendo ACS (Adecuaciones Curriculares).
  </Card>
  
  <Card title="📝 Rúbricas" icon="pencil">
    Crea rúbricas de evaluación detalladas con criterios y niveles de desempeño personalizables.
  </Card>
  
  <Card title="⚖️ Conducta" icon="document">
    Gestiona el comportamiento estudiantil según normativa MEP (REA 40862-V21).
  </Card>
  
  <Card title="✅ Asistencia" icon="list-format">
    Control diario de asistencia con alertas automáticas y reportes estadísticos.
  </Card>
  
  <Card title="📊 Reportes" icon="information">
    Genera reportes profesionales en PDF y Excel listos para presentación oficial.
  </Card>
  
  <Card title="🔐 Permisos Avanzados" icon="setting">
    Sistema de 168 permisos granulares para control de acceso preciso.
  </Card>
  
  <Card title="🇨🇷 Cumplimiento MEP" icon="approve-check">
    Diseñado específicamente para cumplir con las regulaciones del MEP de Costa Rica.
  </Card>
</CardGrid>
